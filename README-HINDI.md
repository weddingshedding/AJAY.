# AJAY SAINI PORTFOLIO — GitHub Pages Ready

Ye version direct GitHub Pages ke liye bana hai. Isme npm, Next.js build ya command ki zaroorat nahi hai.

## Sahi upload process

1. Purane `AJAY` repository ke saare old website files delete karo.
2. Is ZIP ko computer me Extract All karo.
3. Extracted folder ke andar ki saari files select karo:
   - index.html
   - style.css
   - script.js
   - 404.html
   - manifest.webmanifest
   - .nojekyll
   - assets folder
4. GitHub repository me `Add file > Upload files` se root me upload karo.
5. `Commit changes` dabao.
6. Settings > Pages me:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /(root)
7. Save karo. 1–3 minute baad link refresh karo.
